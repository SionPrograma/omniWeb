# OmniWeb Chips
Estructura modular de funcionalidades ("chips") independientes.

- `chip-reparto/`: Sistema de logística (Portable First).
- `chip-finanzas/`: Control financiero personal.
- `chip-idiomas-ia/`: Aprendizaje de idiomas asistido por IA.
- `chip-programacion/`: Educación en arquitectura y desarrollo.
- `chip-musica/`: Gestión de práctica y teoría musical.
