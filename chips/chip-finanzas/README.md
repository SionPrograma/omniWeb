# Chip Finanzas
Módulo para el control financiero personal.

## Estructura
- `dashboard/`: Vista principal con KPIs.
- `ingresos/`: Registro de entradas de dinero.
- `gastos/`: Registro de salidas.
- `ahorro/`: Seguimiento de fondos de reserva.
- `deudas/`: Gestión de pasivos.
- `objetivos/`: Metas de ahorro/compra.
- `historial/`: Registro histórico de movimientos.
- `configuracion/`: Ajustes de moneda y preferencias.
- `shared/`: Utilidades financieras comunes.
