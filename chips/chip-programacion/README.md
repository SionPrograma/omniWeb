# Chip Programación
Módulo enfocado en la enseñanza de arquitectura de software y programación.

## Estructura
- `fundamentos/`: Conceptos básicos y teóricos.
- `logica/`: Algorítmica y razonamiento lógico.
- `arquitectura/`: Patrones de diseño y sistemas de software.
- `tecnologias/`: Lenguajes y frameworks.
- `herramientas/`: IDEs, Git, CI-CD.
- `proyectos/`: Guías para proyectos reales.
- `ejercicios/`: Retos interactivos.
- `mentor-mode/`: Tutoría basada en IA.
- `shared/`: Recursos de formación comunes.
