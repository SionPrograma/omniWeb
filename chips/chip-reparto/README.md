# Chip Reparto
Sistema de logística (PORTABLE FIRST / MAP OPTIONAL).

## Estructura
- `core/`: Lógica de negocio (reparto, entregas, estados).
- `frontend/`: Interfaz para dispositivos (interfaz portable).
- `map/`: Visualización geográfica (Map Optional).
- `shared/`: Recursos comunes.

*Principio: El portable controla la lógica; el mapa solo visualiza.*
