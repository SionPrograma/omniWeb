# Chip Idiomas-IA
Módulo para el aprendizaje de idiomas con inteligencia artificial.

## Estructura
- `input/`: Entrada de texto y audio.
- `transcription/`: Pipeline de modelos ( Whisper/Librosa ).
- `translation/`: Modelos de traducción.
- `phonetics/`: Procesamiento fonético.
- `bilingual-output/`: Generación de texto estructurado bilinguüe.
- `playback/`: Reproducción y TTS.
- `learning-mode/`: Actividades de práctica.
- `shared/`: Diccionarios y lógica común.

*Basado en el módulo legacy `modules/lingua`.*
