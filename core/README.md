# OmniWeb Core
Capa base del sistema para navegación, gestión de estado y almacenamiento compartido.

- `navigation/`: Gestiona la navegación global entre chips.
- `state/`: Gestor de estado para persistencia y restauración de sesiones.
- `storage/`: Abstracción de almacenamiento local/remoto.
- `shared/`: Recursos compartidos (estilos, utilidades).
