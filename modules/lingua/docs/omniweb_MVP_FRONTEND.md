# Frontend Guide 🎨

## Design System
- **Theme**: Dark Tech.
- **Glassmorphism**: 12px blur, 0.7 opacity.
- **Typography**: Outfit for UI, JetBrains Mono for technical data.

## Modules
- `api.js`: Handles fetch/polling.
- `ui.js`: Manages visibility and progress values.
- `app.js`: Main state orchestrator.

## Customization
To change the primary color, update `--accent-color` in `css/style.css`.
